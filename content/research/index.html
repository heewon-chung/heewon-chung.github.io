Research

My research interests broadly encompass cryptography, with a particular emphasis on privacy-enhancing technologies and the design of efficient cryptographic protocols for real-world applications.

Zero-Knowledge Proofs & Verifiable Computation
Zero-knowledge proofs enable proving statements without revealing underlying information, providing succinct and efficient verification. I focus on developing practical zero-knowledge proof systems like Bulletproofs+ that achieve both efficiency and security. These techniques are fundamental to blockchain scalability and privacy-preserving applications.
Blockchains & Privacy-Preserving Transactions
I work on improving blockchain scalability and privacy by leveraging zero-knowledge proofs and verifiable computation. My research on Bulletproofs+ was adopted by Monero in August 2020 to enhance the efficiency of confidential transactions, significantly reducing transaction sizes while preserving strong privacy guarantees. This bridges the gap between cryptographic theory and practical blockchain deployment.
Secure Multi-Party Computation (MPC)

Secure multiparty computation enables multiple parties to jointly compute functions without revealing individual inputs. I research efficient cryptographic protocols that enable privacy-preserving collaborative computation across diverse applications.

Private Set Operations (PSO)
Private set operations allow computing set intersections, unions, and other operations on private datasets without revealing individual elements. This is essential for privacy-preserving data analysis and collaborative filtering applications.
Private Information Retrieval (PIR)
Private information retrieval enables querying databases without revealing which item was requested to the database server. This protects user privacy in information retrieval systems and has applications in secure search and recommendation systems.
Function Secret Sharing (FSS)
Function secret sharing splits functions into secret shares that can be distributed across multiple parties for evaluation without revealing the function itself. This enables efficient distributed computation and privacy-preserving function evaluation.
Fully Homomorphic Encryption (FHE)
Fully homomorphic encryption enables computation on encrypted data without decryption, allowing cloud computing and privacy-preserving data analysis. My Ph.D. research focused on FHE and its practical applications, exploring techniques to improve efficiency and expand applicability across various domains including secure cloud computing and privacy-preserving machine learning.
